# 🎓 Student Performance Analyser

A Python-based data analysis mini project that uses NumPy to evaluate and analyze student academic performance across multiple subjects.

## 📌 Mini Project Overview

This mini project demonstrates how NumPy can be used for educational data analysis. It processes marks obtained by students in different subjects and performs various analytical operations such as calculating averages, identifying passing students, normalizing data, and applying bonus marks using broadcasting techniques.

## 🚀 Features

- Calculate average marks for each student
- Calculate average marks for each subject
- Identify students who passed based on average score
- Normalize marks using Min-Max Normalization
- Apply bonus marks using NumPy Broadcasting
- Demonstrate efficient matrix-based computations

## 🛠️ Technologies Used

- Python
- NumPy
- Jupyter Notebook

## 📊 Analysis Performed

### 1. Student-wise Average
Calculates the average marks obtained by each student across all subjects.

### 2. Subject-wise Average
Determines the overall performance of each subject.

### 3. Pass/Fail Evaluation
Identifies students whose average score is greater than or equal to 60.

### 4. Data Normalization
Converts marks into a range between 0 and 1 using Min-Max Scaling.

### 5. Bonus Marks Application
Applies additional marks to Mathematics using NumPy broadcasting.
